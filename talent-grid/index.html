📁 如何新增部門？（不需要寫程式）

將各部門的 Excel 評核表上傳給 Claude，請他產生對應的九宮格 HTML 檔

把新的 HTML 檔與本首頁（index.html）放在同一個資料夾中

請 Claude 在首頁新增該部門的卡片（提供部門名稱與人數即可）

將整個資料夾放到公司內部網路（NAS / SharePoint），主管即可瀏覽
